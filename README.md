# What is this?
This application is a Web API built on .Net Core. This repository has also a website built was created using Angular and the style with Bootstrap

# Run Web API
- Load the **WebApplication.sln** file that is in the backend folder in your Visual Studio.
- After loading the application into Visual Studio, run the project build and then start the application.

# Run Website
- Open the folder Frontend and run the command `npm install` and then `npm start`
